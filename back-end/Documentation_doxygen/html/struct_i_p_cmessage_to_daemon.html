Detailed Description

Struct contains the source, destination, the task and the type of job.

Parameters
sourceChar array with len of 1024, contains the source from where it should copy.
destinationChar array with len of 1024, contains the destination to where it should copy.
taskChar array with len of 1024, tells the deamon what job it should accomplish: -"copy" = tells the deamon to start a copy tipe job -"status" = gets the status of a current job -"stop" = stops a job -"suspent" = pauses / suspends a job -"resume" = resumes a job
Warning
A wrong typed task will result in the deamon doing nothing.
Parameters
jobIDInteger, tells us the thread id.
